# <img src="https://raw.githubusercontent.com/k10xp/game_score_frontend/refs/heads/main/public/sportz-scores-logo.svg" alt="Project logo" width="50"> SomeName Inc. Sportz Scores

Fullstack web application (clean frontend backend split) to log sports game results.

User can

- enter a game score (date, home team, away team, final score)
- view all game score (winning team highlight in green, losing team in red)

See [design](./design-doc/base.md) for more info.

## Instructions

Format code with prettier

```shell
npm run fmt
```

Run vue app

```shell
npm run dev
```

Format before run vue app

```shell
npm run dev1
```
